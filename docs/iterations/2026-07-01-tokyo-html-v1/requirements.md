# 本轮需求：东京旅行模板 v1.0

> 状态：源需求材料。后续 spec 应读取聚合后的 `docs/iterations/2026-07-01-v1.0-travel-requirements/requirements.md`。

1. 旅行模板列表展示封面、标题、`desc`、天数和标签。
2. 原生模板详情提供 `浏览完整攻略` 和 `使用此计划`。
3. 完整攻略通过 `web-view` 只读展示现有美观 HTML。
4. 用户使用模板后生成独立、可编辑的空间实例。
5. 实例支持节点和模块的增删改排、图片、地图、任务与预算。
6. 模板、H5 和实例创建必须共享同一个版本化模板数据源。
7. H5 不可用时不阻塞实例创建。
8. 访客只读，敏感确认字段默认隐藏。

验收标准见 `docs/versions/v1.0/travel-prd.md`。
